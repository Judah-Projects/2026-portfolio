Judah's Portfolio

This is a collection projects that I’ve been developing as I learn the core skills of modern web creation. Through these builds, I’ve practiced structuring pages with HTML, styling and layout with CSS, adding interactivity with JavaScript, exploring UX and research, designing user friendly experiences, creating logos and brand identities, debugging and troubleshooting, and working with generative AI to speed up ideation and development. Each project represents a different part of the learning journey — from design thinking to hands on coding — and together they show the range of tools I’m becoming comfortable with.

Selected Projects

Caring Kitchen

My passion project is about feeding our future.

I built and designed this entirely by hand with HTML, CSS, JavaScript, and a little help from AI. I created the logo with Canva. The concepts in the copy have been evolving for decades though I have only been sharing and photographing for a few years. The aim of the project is simple: save money, time, and health by practicing self-care today that will feed you and your loved ones in the future.
